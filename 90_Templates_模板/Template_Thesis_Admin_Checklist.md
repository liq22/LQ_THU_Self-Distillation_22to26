# 博士毕业手续清单模板

## 阶段清单

| 阶段 | 材料 | 来源 | 状态 | 备注 |
|---|---|---|---|---|
| 图像采集 | 采集码/照片/学籍核对 | 学校系统 | 待办/完成 |  |
| 送审前 | 论文格式、学术规范、送审材料 | 学院/研究生院 | 待办/完成 |  |
| 评阅后 | 评阅意见答复、修改说明 | 导师/学院 | 待办/完成 | 不提交原始意见 |
| 答辩申请 | 答辩公告、委员会材料、决议书模板 | 学院/秘书 | 待办/完成 | 不记录个人名单 |
| 申报学位 | 学位申请、成绩、评阅材料、情况简介 | 多系统 | 待办/完成 | 原件私有 |
| 答辩后 | 最终论文、授权、系统确认、归档 | 学校系统 | 待办/完成 |  |

## 命名建议

- `阶段-材料名-状态`
- 状态可用：`待填`、`待签字`、`已签字`、`已扫描`、`已提交`、`最终版`

## GitHub 版检查

- [ ] 不包含学号、联系方式、签字页、成绩、评阅意见、评委名单。
- [ ] 不包含论文全文和扫描 PDF。
- [ ] 只保留流程、材料类型和复盘经验。
