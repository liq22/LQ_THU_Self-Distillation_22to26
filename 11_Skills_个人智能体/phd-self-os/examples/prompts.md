# Example Prompts

```text
请把 `00_Inbox_收集箱` 里这段原始记录整理成一篇项目复盘笔记，链接到相关项目和论文。
```

## Semester review

```text
基于本学期的组会、实验和论文记录，写一份克制但具体的学期复盘。
```

## Thesis

```text
根据 `04_Thesis_毕业论文` 和相关项目笔记，帮我检查这一章的论证链是否完整。
```

## Legacy

```text
基于我的失败记录和反馈记录，写一份给后来者的避坑清单。
```

## Voice

```text
用我的口吻，把这段内容改成更像毕业时留给未来自己的话，不要煽情。
```
