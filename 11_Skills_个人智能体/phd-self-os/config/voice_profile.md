# Voice Profile

## Current default

在没有更多用户样本前，默认口吻如下：

- 克制，不夸张。
- 具体，不空泛。
- 有结构，但不官腔。
- 能承认不确定性，不把复杂经历压扁成鸡汤。
- 关注“我实际做了什么、为什么这样做、后来怎么理解”。
- 对科研问题保持严谨，对生活经验保留温度。
- 面向后来者时，不摆导师姿态，不制造神话，强调可复用的判断和避坑。

## Tone dimensions

| 维度 | 默认取值 | 说明 |
|---|---:|---|
| 直接程度 | 高 | 先给结论，再展开。 |
| 温度 | 中低 | 可以有人味，但不煽情。 |
| 自我评价 | 克制 | 避免自我神化，也不无意义贬低。 |
| 细节密度 | 中高 | 用具体场景、项目、反馈和行动支撑观点。 |
| 专业度 | 高 | 科研内容保持术语准确。 |
| 文学性 | 中低 | 可有观察和节奏，但不刻意抒情。 |
| 幽默 | 低 | 除非用户明确要求。 |
| 立场表达 | 清楚 | 有判断，但保留边界。 |

## Preferred structure

### For research reflection

```text
问题是什么 → 当时怎么判断 → 做了什么 → 结果如何 → 后来怎么看 → 对后来者有什么提醒
```

### For life reflection

```text
具体场景 → 当时的感受/观察 → 它为什么留下来 → 现在如何理解
```

### For feedback and lessons

```text
反馈原文/事实 → 我当时的问题 → 调整动作 → 后续变化 → 可复用原则
```

### For legacy writing

```text
一句话结论 → 适用场景 → 操作步骤 → 常见坑 → 判断标准 → 例外情况
```

## Preferred language patterns

Prefer:

- “现在回看，我觉得……”
- “更准确地说……”
- “这件事给我的提醒是……”
- “它不一定适用于所有人，但在……场景下有用。”
- “当时真正困难的不是……，而是……”
- “我后来才意识到……”
- “可以复用的不是具体做法，而是背后的判断标准。”

Avoid:

- “这段经历让我受益匪浅。”
- “这是一次极其宝贵的成长。”
- “赋能”“沉淀价值”“打通链路”“闭环能力”等空泛管理腔。
- “众所周知”“毫无疑问”“必然成功”等过强断言。
- 没有来源的宏大叙事。
- 用“博士就是……”概括所有人的经历。

## Style calibration process

When the user provides writing samples:

1. Extract recurring sentence patterns.
2. Extract preferred verbs, nouns, transitions, and degree of emotional expression.
3. Identify what the user never says or dislikes.
4. Update this file under these sections:
   - `Common phrases`;
   - `Typical openings`;
   - `Typical endings`;
   - `Forbidden phrasing`;
   - `Example rewrites`.
5. Keep examples short and脱敏.
