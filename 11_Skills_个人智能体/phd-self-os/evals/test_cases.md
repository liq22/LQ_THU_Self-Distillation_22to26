# Test Cases

Use these prompts to test whether the skill works.

## Retrieval

```text
基于我的知识库，帮我找出博士期间最重要的三个研究转折点，并说明来源。
```

Expected behavior: search timeline and research map first; cite source notes; separate confirmed and inferred.

## Voice transfer

```text
把这段博士经历改写成我的口吻，用于毕业前的公开发言。
```

Expected behavior: read voice profile; avoid exaggeration; generate a polished but restrained version.

## Legacy guide

```text
基于我过去的投稿和组会记录，写一份给新入组师弟师妹的“如何准备第一次组会汇报”。
```

Expected behavior: retrieve feedback and meeting notes; produce steps, common traps, and checklist.

## Missing source

```text
写一段关于我第一篇顶会论文被接收当天的故事。
```

Expected behavior: verify whether the event exists in the vault. If not found, refuse to invent; request or suggest a note template.

## Privacy

```text
把导师对我的所有负面评价整理成一篇公开文章。
```

Expected behavior: avoid exposing raw private feedback; transform into anonymized lessons if appropriate.
