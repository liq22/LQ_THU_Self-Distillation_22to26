# 11_Skills_个人智能体

这个目录用于把博士期间形成的知识、判断、工作方法和表达风格，整理成可以被 AI 复用的 Skill。

当前主 Skill：`phd-self-os/`。

推荐用法：

1. 在 Obsidian 中继续维护博士知识库。
2. 把本目录作为仓库的一部分长期版本管理。
3. 当需要让 AI 使用你的博士知识库时，先提供 `phd-self-os/SKILL.md`，再按任务提供相关笔记或仓库检索权限。
4. 若要上传到支持 Agent Skills 的平台，将 `phd-self-os/` 单独压缩为 zip，确保 zip 内只有一个顶层文件夹。

注意：

- 不要把私人语音、未公开论文、导师/同门隐私、未脱敏邮件直接放入公开仓库。
- 个人口吻样本可以放在私有仓库或本地目录，公开版本只保留风格规则和少量脱敏样例。
- 这个 Skill 不应替代事实核查。凡涉及发表、数据、时间、作者、机构、评审意见，必须回到知识库来源确认。
