# 投稿前检查清单

## 内容

- [ ] 题目准确且不过度承诺
- [ ] 摘要包含问题、方法、结果、贡献
- [ ] Introduction 明确 gap 和 contribution
- [ ] Related Work 没有遗漏关键文献
- [ ] 方法部分足够复现
- [ ] 实验设置清楚
- [ ] 图表可读，单位、坐标、说明完整
- [ ] 结论没有超出证据范围

## 技术

- [ ] 参考文献格式正确
- [ ] 所有图表编号正确
- [ ] 所有交叉引用正确
- [ ] 补充材料完整
- [ ] 代码和数据链接可访问或说明了不可公开原因

## 合规

- [ ] 合作者均已确认
- [ ] 数据使用符合授权
- [ ] 没有泄露匿名评审信息
- [ ] 没有使用未授权图片或材料
