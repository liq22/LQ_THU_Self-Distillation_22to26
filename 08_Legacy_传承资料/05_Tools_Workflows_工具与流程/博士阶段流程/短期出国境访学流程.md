# 短期出国境访学流程

## 摘要

短访材料包括申请、录取后手续、离校/返校、报销和总结报告。GitHub 版本保留流程和模板，不上传邀请信、资助协议、报销表、外方导师评语或个人总结原件。

## 流程总表

| 阶段 | 目标 | 典型材料 | 注意点 |
|---|---|---|---|
| 申请前 | 判断是否符合基金条件 | 申请指南、培养基金规定、邀请信要求 | 先确认时间长度、目的地和导师支持 |
| 申请材料 | 提交访学基金申请 | 研究计划、邀请信、申请自查表 | 研究计划要和博士主线一致 |
| 录取后 | 完成派出前手续 | 录取提示、出国境须知、资助协议 | 关注离校、保险、经费和变更规则 |
| 访学期间 | 记录工作和交流 | 访学日志、seminar 记录、合作进展 | 每周记录，避免回国后补写困难 |
| 回校报到 | 完成返校确认 | 临时离校/回校手续单 | 及时确认系统状态 |
| 报销与总结 | 完成经费和成果闭环 | 收支情况表、总结报告、外方导师评语 | 票据、汇率、资助标准要单独核对 |

## 总结报告模板

1. 访学背景：访问单位、研究方向和出访目的。
2. 主要工作：按时间或主题列出完成事项。
3. 学术交流：seminar、组会、合作讨论、课程或 workshop。
4. 研究收获：对博士主线的具体帮助。
5. 生活与适应：只写可共享经验，不记录私人细节。
6. 后续计划：论文、合作、数据、代码或回访安排。
7. 给后来者的建议：申请、出行、沟通、报销。

## 录取后 Checklist

- [ ] 保存录取通知和资助标准。
- [ ] 确认是否需要改派、延期或变更行程。
- [ ] 完成离校/出国境相关手续。
- [ ] 建立票据与报销记录表。
- [ ] 与访问导师确认抵达、汇报和研究计划。
- [ ] 回校后尽快完成报到、总结报告和报销。

## GitHub 上传边界

- 可提交：流程说明、总结报告模板、脱敏复盘。
- 不提交：邀请信、护照/签证、资助协议、票据、报销表、导师评语、个人行程细节。
