# 培养环节材料索引

## 摘要

博士培养环节材料包括资格考试、选题报告、中期检查、创新成果审核、最终学术报告和学术活动记录。GitHub 版本只保留材料类型、准备顺序和 Markdown 模板，不上传 Office 原件。

## 环节总表

| 环节 | 典型材料 | 准备重点 | 上传边界 |
|---|---|---|---|
| 资格考试 | 考试记录、汇报材料、路线图 | 研究问题定义和基础知识 | 不上传题库和个人 PPT |
| 选题报告 | 申请表、论文工作计划、实验安全分析 | 研究问题、路线、风险 | 不上传签字前原件 |
| 中期检查 | 年度/中期考核表、论文进展表 | 已完成工作和后续计划 | 不上传含评价表格 |
| 创新成果审核 | 创新成果报告审核表 | 成果和学位要求匹配 | 只保留清单 |
| 最终学术报告 | 申请表、记载表 | 报告时间、材料提交、记录归档 | 只保留流程 |
| 学术活动 | 学术活动记录表 | 活动名称、时间、角色、证明 | 不上传证明原件 |

## 通用准备流程

1. 从学院/研究生院确认最新版要求。
2. 建立私有材料夹，原件按“环节-材料名-状态”命名。
3. 建立公开版 Markdown 索引，只记录材料类型和经验。
4. 所有签字、评价、成绩、评审意见和证明扫描件留在本地私有区。
5. 每完成一个环节，补一条复盘：准备周期、最容易遗漏的材料、下次如何更快。

## GitHub 上传边界

- 可提交：环节说明、材料清单、准备经验、空白 Markdown 模板。
- 不提交：Office 原件、签字页、评价意见、成绩、学号、证明扫描件、导师或评委个人信息。
