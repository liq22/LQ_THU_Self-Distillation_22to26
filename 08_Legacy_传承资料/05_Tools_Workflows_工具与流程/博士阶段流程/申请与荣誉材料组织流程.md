# 申请与荣誉材料组织流程

## 摘要

用于整理水木学者、国家奖学金、青年人才项目等申请材料。GitHub 版本保留组织方法、材料清单和答辩模板，不上传申请表原件、评分表、证明照片、推荐意见、论文 PDF 和个人证书扫描件。

## 材料组织结构

| 类别 | 典型材料 | 建议处理 |
|---|---|---|
| 基本申请 | 申请表、个人简介、研究计划 | 原件私有，公开版提炼结构 |
| 推荐/审核 | 院系推荐意见、导师意见 | 不提交 |
| 学术成果 | 论文、专利、项目、会议 | 只提交成果索引，不提交全文 PDF |
| 综合素质 | 社会实践、体育、服务、荣誉 | 只保留脱敏条目 |
| 支撑材料 | 证书、通知、照片、网页截图 | 原件私有，公开版记录材料类型 |
| 答辩展示 | PPT、讲稿、Q&A | 提炼页结构和准备经验 |

## 准备流程

1. 先读评审办法，拆出评分项、硬性门槛和提交格式。
2. 建一张成果台账，所有支撑材料都从台账反查。
3. 先准备“可被核验的事实”，再写个人叙事。
4. 答辩 PPT 只放最能支撑评审标准的证据，不堆材料。
5. 申请结束后补复盘：哪些材料最耗时，哪些可提前维护。

## GitHub 上传边界

- 可提交：材料组织方法、成果台账模板、答辩结构、经验复盘。
- 不提交：评分表、申请审批表、推荐意见、证书照片、网页截图、论文全文、个人联系方式、学号和排名细节。
