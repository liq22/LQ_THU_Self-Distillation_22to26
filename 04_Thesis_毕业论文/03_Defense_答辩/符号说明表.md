## 泛化界中的符号

### 基本公式
```
R(h) ≤ R̂_S(h) + 2R̂_S(H) + √(ln(1/δ)/(2n))
```

### 符号说明

| 符号 | 含义 | 直观解释 |
|------|------|----------|
| h | 某一个具体模型/假设函数 | 训练后得到的一个诊断模型 |
| H | 假设空间 | 所有可能模型的集合 |
| R(h) | 真实风险 / 期望风险 | 模型在未知总体分布上的真实误差 |
| R̂_S(h) | 经验风险 | 模型在训练样本集 S 上的误差 |
| S | 训练样本集 | S = {(x_i, y_i)}_(i=1 to n) |
| n | 样本数量 | 样本越多，泛化误差项通常越小 |
| δ | 置信失败概率 | 泛化界以至少 1-δ 的概率成立 |
| ln(1/δ) | 置信惩罚项 | 要求越高置信度，惩罚越大 |
| R̂_S(H) | 经验 Rademacher 复杂度 | 衡量假设空间拟合随机噪声的能力 |
| √(ln(1/δ)/(2n)) | 统计置信项 | 来自有限样本带来的概率误差 |

### 一句话解释
**泛化误差 = 训练误差 + 模型复杂度惩罚 + 有限样本置信惩罚。**

这里图的重点不是降低 R̂_S(h)，而是解释为什么 NSNet 可以降低 R̂_S(H)。

## 稀疏结构项中的符号

### 公式
```
|Z|_0 ≤ k ≪ M
|A| ≤ Σ(j=0 to k) C(M,j) ≤ (eM/k)^k
log|A| = O(k·log M)
```

### 符号说明

| 符号 | 含义 | 直观解释 |
|------|------|----------|
| Z | 离散结构变量 / 结构选择向量 | 表示哪些算子、节点或连接被激活 |
| \|Z\|_0 | ℓ_0 稀疏度 | 被激活的结构单元数量 |
| M | 候选结构单元总数 | 所有可选节点、算子或连接的数量 |
| k | 最大激活结构数 | 推理阶段最多只允许激活 k 个结构单元 |
| k ≪ M | 稀疏条件 | 激活结构远少于候选结构 |
| A | 离散架构集合 | 所有可能的稀疏拓扑结构集合 |
| a | 一个具体离散架构 | A 中的某一个结构选择 |
| \|A\| | 架构集合规模 | 可选模型结构的总数量 |
| j | 激活结构数量索引 | 从 0 到 k 枚举不同稀疏度 |
| C(M,j) | 从 M 个结构中选 j 个的组合数 | 激活 j 个结构单元的可能选择数 |
| Σ(j=0 to k) C(M,j) | 最多选 k 个结构的总组合数 | 稀疏结构搜索空间大小 |
| e | 自然常数 | 组合数上界中的常数 |
| log \|A\| | 架构集合的对数容量 | 复杂度分析中通常看对数规模 |
| O(k·log M) | 离散结构复杂度阶 | 表示稀疏拓扑的复杂度主项 |

### 一句话解释
**如果不加稀疏约束，结构搜索可能接近指数级；加入 \|Z\|_0 ≤ k 后，结构复杂度被压缩到 k·log M 量级。**

## 参数覆盖项中的符号

### 公式
```
Ω_eff ⊂ R^(d_eff)
N(Ω_eff, \|·\|_2, ρ) ≤ (3C_eff/ρ)^(d_eff)
```

### 符号说明

| 符号 | 含义 | 直观解释 |
|------|------|----------|
| Ω_eff | 有效连续参数空间 | 物理参数和判别参数共同形成的受限可行域 |
| R^(d_eff) | d_eff 维欧氏空间 | 有效参数所在的连续空间 |
| d_eff | 有效参数维度 | 真正需要搜索的连续自由度数量 |
| C_eff | 有效参数空间直径或尺度上界 | 参数可行域的大小 |
| ρ | 参数空间覆盖半径 | 用半径为 ρ 的小球覆盖参数空间 |
| \|·\|_2 | 欧氏范数 | 衡量两个参数点之间的距离 |
| N(Ω_eff, \|·\|_2, ρ) | 参数空间覆盖数 | 用多少个半径为 ρ 的小球可以覆盖参数空间 |
| (3C_eff/ρ)^(d_eff) | 覆盖数上界 | 维度越高、空间越大、半径越小，覆盖数越大 |

### 一句话解释
**物理约束把参数限制在 Ω_eff 内，使搜索不再发生在无约束高维空间，而是发生在较低维、有限尺度的可行域中。**

## Lipschitz 映射中的符号

### 公式
```
d_S(F_(ϑ_1,a), F_(ϑ_2,a)) ≤ L_eff\|ϑ_1 - ϑ_2\|_2
N(H_a, d_S, ε) ≤ (3C_eff·L_eff/ε)^(d_eff)
```

### 符号说明

| 符号 | 含义 | 直观解释 |
|------|------|----------|
| ϑ | 联合连续参数 | 通常包含物理参数与判别参数 |
| ω | 物理参数 | 如中心频率、带宽、尺度、阈值等 |
| η | 非物理判别参数 | 分类器、门控或判别层中的可学习参数 |
| ϑ = (ω, η) | 有效参数向量 | 物理参数与判别参数的合并表示 |
| F_(ϑ,a) | 给定参数和结构后的模型函数 | 固定架构 a、参数为 ϑ 的 NSNet |
| F_(ϑ_1,a) | 参数为 ϑ_1 的模型 | 同一个架构下的一个模型实例 |
| F_(ϑ_2,a) | 参数为 ϑ_2 的模型 | 同一个架构下的另一个模型实例 |
| d_S(·,·) | 样本集 S 上的经验函数距离 | 衡量两个模型在训练样本上的输出差异 |
| L_eff | 有效 Lipschitz 常数 | 参数变化会引起多大函数输出变化 |
| ε | 函数空间覆盖半径 | 在函数输出层面允许的近似误差 |
| H_a | 固定架构 a 下的函数类 | 只改变连续参数、不改变结构时的模型集合 |
| N(H_a, d_S, ε) | 固定架构下的函数覆盖数 | 用多少个代表函数可以覆盖该函数类 |

### 一句话解释
**如果参数到函数的映射足够稳定，即满足 Lipschitz 条件，那么参数空间的小覆盖可以传递为函数空间的小覆盖。**

## 并集上界与最终复杂度界中的符号

### 公式
```
H_NSNet = ∪(a∈A) H_a
log N(H_NSNet, d_S, ε) ≤ O(k·log M) + d_eff·log(3C_eff·L_eff/ε)
```

### 最终建议写成
```
R̂_S(H_NSNet) ≤ O(√(k·log M/n) + √(d_eff·log(C_eff·L_eff·n)/n))
```

### 符号说明

| 符号 | 含义 | 直观解释 |
|------|------|----------|
| H_NSNet | NSNet 推理阶段假设空间 | 所有可能 NSNet 模型的集合 |
| ∪(a∈A) H_a | 所有固定架构函数类的并集 | 每个架构 a 对应一个子函数类，全部合起来就是整体 NSNet 假设空间 |
| N(H_NSNet, d_S, ε) | 整体函数类覆盖数 | 覆盖整个 NSNet 假设空间需要多少代表函数 |
| log N(·) | 覆盖数的对数 | 常用于复杂度界推导 |
| O(k·log M) | 结构复杂度项 | 来自稀疏拓扑选择 |
| d_eff·log(3C_eff·L_eff/ε) | 参数复杂度项 | 来自连续参数空间覆盖 |
| R̂_S(H_NSNet) | NSNet 的经验 Rademacher 复杂度 | 图中最终希望降低的复杂度项 |
| √(k·log M/n) | 稀疏结构项 | 结构越稀疏、样本越多，该项越小 |
| √(d_eff·log(C_eff·L_eff·n)/n) | 有效参数项 | 有效维度越低、样本越多，该项越小 |

### 一句话解释
**NSNet 的经验复杂度被拆成两项：第一项来自"选哪些结构"，第二项来自"每个结构下调哪些参数"。**

## 最推荐保留的核心解释

最终图中最推荐保留的核心解释是：

```
      √(k·log M/n)          √(d_eff·log(C_eff·L_eff·n)/n)
         ↓                           ↓
   结构项：稀疏拓扑              参数项：有效自由度
```

即：**稀疏结构降低可选路径数，物理参数约束降低有效自由度，二者共同压缩经验复杂度项。**
