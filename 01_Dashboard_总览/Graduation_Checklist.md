# Graduation Checklist

## 科研归档

- [ ] 每个研究项目都有项目页
- [ ] 每个项目都有核心结论、失败记录和后续建议
- [ ] 关键代码仓库有 README
- [ ] 关键数据有来源、权限、处理流程说明
- [ ] 未发表内容已标注 `visibility: private`

## 论文归档

- [ ] 每篇论文有状态页
- [ ] 投稿、审稿、rebuttal 已整理或脱敏
- [ ] 已发表论文记录 DOI、链接、代码、数据
- [ ] 文献笔记按主题归档

## 毕业论文

- [ ] 大纲完成
- [ ] 各章节材料归位
- [ ] 图表列表完成
- [ ] 答辩材料完成
- [ ] 学校手续清单完成

## 传承资料

- [ ] 新人入门指南
- [ ] 实验室常用流程
- [ ] 工具链说明
- [ ] 常见坑点 FAQ
- [ ] 推荐阅读和学习路径

## 隐私与公开

- [ ] 删除或脱敏个人信息
- [ ] 删除未授权 PDF
- [ ] 删除敏感审稿意见
- [ ] 删除未授权数据
- [ ] 确认仓库 public/private 状态
