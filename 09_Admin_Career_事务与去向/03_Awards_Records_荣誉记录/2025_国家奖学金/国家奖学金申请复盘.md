# 2025 国家奖学金申请复盘

## 摘要

这份复盘记录李奇准备国家奖学金申请的材料组织方法和答辩准备经验。GitHub 版本不保存申请审批表、计分表、证明照片、证书扫描件、论文全文 PDF、排名分数和个人编号。

## 材料组织方法

| 模块 | 准备重点 | 可上传版本 |
|---|---|---|
| 申请表 | 事实准确、成果分类清楚 | 只保留填写策略 |
| 计分表 | 每一项都有支撑材料 | 保留台账模板，不保留分数 |
| 学术成果 | 论文、专利、会议按规则归类 | 保留成果目录，不上传全文 |
| 综合素质 | 社会实践、体育、服务等分类 | 保留脱敏条目 |
| 答辩展示 | 用少量证据支撑核心叙事 | 保留 PPT 结构 |

## 准备顺序

1. 先拆评审办法和计分细则，明确哪些材料必须准备。
2. 建支撑材料台账，给每个证明材料编号。
3. 申请表、计分表和支撑材料互相校验，避免事实不一致。
4. 答辩 PPT 只呈现最能说明问题的成果，不把材料包搬上去。
5. 申请结束后记录耗时环节，为下一次申请复用。

## 答辩 PPT 结构

- 个人主线：博士阶段研究问题和成长轨迹。
- 学术成果：挑最能代表贡献的成果，说明角色和影响。
- 综合表现：社会实践、服务、交流和集体贡献。
- 未来计划：奖学金对后续研究和毕业工作的支撑。
- Q&A 准备：成果真实性、贡献边界、计分依据、未来安排。

## 经验教训

- 支撑材料要平时维护，临近截止再找会非常耗时。
- 成果叙事要服务评审标准，不要把 CV 逐条复述。
- 计分规则要逐条核对，模糊项单独标注依据，避免答辩时解释不清。

## GitHub 上传边界

- 可提交：材料组织方法、答辩结构、脱敏复盘。
- 不提交：原始申请表、计分表、证书照片、截图、论文全文、个人编号、分数排名。
