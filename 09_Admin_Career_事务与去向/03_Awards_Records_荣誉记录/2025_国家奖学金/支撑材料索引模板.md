# 支撑材料索引模板

## 用途

用于整理奖学金、人才项目和荣誉申请的证明材料。公开或内部共享版本只保留材料类型，不附原件。

| 编号 | 评审项 | 材料名称 | 证明什么 | 原件位置 | 是否可公开 | 备注 |
|---|---|---|---|---|---|---|
| A-01 | 学术成果 | 论文/专利/会议条目 | 成果和角色 | 本地私有区 | 否/可链接公开页面 |  |
| B-01 | 社会实践 | 实践证明/获奖证明 | 实践参与和结果 | 本地私有区 | 否 |  |
| C-01 | 综合素质 | 活动证明/证书 | 综合表现 | 本地私有区 | 否 |  |

## 检查清单

- [ ] 每个计分项都能对应到至少一个支撑材料。
- [ ] 支撑材料名称和申请表表述一致。
- [ ] 截图、证书、照片不进入 GitHub。
- [ ] 公开论文只链接 DOI/官网页面，不上传全文 PDF。
- [ ] 需要解释的计分项单独写备注。
