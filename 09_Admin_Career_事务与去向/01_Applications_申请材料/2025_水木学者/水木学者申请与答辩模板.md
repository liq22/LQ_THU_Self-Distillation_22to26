# 水木学者申请与答辩模板

## 摘要

此模板来自李奇整理水木学者申请材料和答辩材料的经验。GitHub 版本不保存申请书原件、推荐意见、汇总表、答辩 PPT 原件和个人编号。

## 申请材料结构

| 模块 | 写作目标 |
|---|---|
| 基本信息 | 准确、简洁，和 CV 保持一致 |
| 教育与研究经历 | 强调博士阶段主线和独立性 |
| 代表性成果 | 只选能支撑申请定位的成果 |
| 研究计划 | 明确 1-3 年问题、路线和产出 |
| 合作基础 | 说明平台、导师、团队和资源匹配 |
| 未来发展 | 说明为什么适合该计划 |

## 答辩 PPT 结构

1. 个人定位：一句话说明研究方向和申请目标。
2. 博士主线：用时间线说明问题如何收敛。
3. 代表成果：每个成果说明问题、方法、贡献和影响。
4. 未来计划：问题、技术路线、阶段产出。
5. 平台匹配：为什么该平台能支撑这项工作。
6. 风险与应对：数据、合作、论文、时间安排。

## Q&A 准备

| 问题类型 | 准备方式 |
|---|---|
| 为什么申请这个计划 | 从研究延续、平台匹配、个人发展回答 |
| 未来计划是否可行 | 拆成近期可做、中期突破、长期目标 |
| 代表成果中的个人贡献 | 准备角色边界和具体贡献 |
| 和已有团队/方向的关系 | 说明互补性，不夸大独立资源 |

## GitHub 上传边界

- 可提交：材料结构、答辩结构、准备经验。
- 不提交：申请书原件、推荐意见、汇总表、答辩 PPT、个人编号、联系方式。
